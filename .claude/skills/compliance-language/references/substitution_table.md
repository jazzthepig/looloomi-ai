language. Use when editing API responses, frontend copy, investor materials,
macro briefs, signal feed entries, or any user-facing content.

## Descriptive verbs — safe to use

EN: positioned, ranked, weighted, scored, graded, benchmarked, favored,
disfavored, tilted toward, tilted away from, evaluated, assessed, classified,
categorized, placed

ZH: 定位, 排名, 配置, 评分, 评级, 对标, 偏好, 倾向, 归类, 分类, 评估

## Context verbs — safe ONLY when the object is NOT a trading action

EN: recommend (reading the docs), suggest (reviewing methodology), advise
(consulting a licensed professional)

ZH: 建议（阅读文档）, 推荐（参考方法论）, 咨询（持牌专业人士）

## Macro brief / narrative templates

### Regime commentary (safe patterns)

- "Current regime: RISK_OFF. CIS framework tilts positioning toward defensive assets."
- "当前市场体制：避险。CIS框架将定位倾向于防御性资产。"
- "Momentum shift detected: L1 assets moving from NEUTRAL to OUTPERFORM positioning."
- "检测到动能转换：L1资产从中性转向看好定位。"

### NOT safe (never generate these)

- "Risk-off regime means you should sell crypto and buy bonds."
- "避险体制下应卖出加密资产并买入债券。"
- "Momentum says buy the dip."
- "动能显示应逢低买入。"
