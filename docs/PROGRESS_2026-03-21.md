## 平台架构总览

```
Mac Mini M4 Pro (Qwen3 35B)          Railway (FastAPI + React)
├── cis_v4_engine.py                  ├── src/api/main.py (v0.4.1)
├── cis_scheduler.py (30min)          │   ├── routers/market.py
├── cis_push.py → POST ─────────────→│   ├── routers/cis.py
├── macro_analysis/                   │   ├── routers/intelligence.py
│   ├── fetch_data.py (Binance)       │   ├── routers/vault.py
│   ├── call_lmstudio.py (Qwen3)     │   ├── routers/onchain.py
│   └── macro_push.py → POST ───────→│   └── routers/macro.py
│                                     ├── store.py (Redis + Supabase)
│                                     └── dashboard/ (16 React components)
│                                         ├── App.jsx (single-page scroll)
│                                         ├── MarketDashboard.jsx
│                                         ├── IntelligencePage.jsx
│                                         ├── CISLeaderboard.jsx
│                                         ├── CISWidget.jsx
│                                         ├── MacroBrief.jsx
│                                         └── ...
│
└── Freqtrade (dry-run ready)
    ├── CometCloudMultiFactorV2.py   (纯技术面)
    └── CometCloudStrategy.py        (CIS 集成)
```

**数据流向：**
- CIS 评分：Mac Mini → Railway `/internal/cis-scores` → Upstash Redis (2h TTL) + Supabase (持久化) → 前端
- 宏观简报：Mac Mini (LM Studio) → Railway `/internal/macro-brief` → Redis (12h TTL) → 前端
- 市场数据：Railway 直接拉 CoinGecko + DeFiLlama + yfinance
